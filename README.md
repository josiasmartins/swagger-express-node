# Swagger with express node

## Auto gen swagger doc with express (node)

## How use

#### step 1
```
git clone https://github.com/josiasmartins/swagger-express-node
```

##### step 2. 
Enter in folder > npm i, after execute script above
```
npm run swagger
```
